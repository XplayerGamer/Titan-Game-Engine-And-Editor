Titan Editor
======

This is the Titan Editor, a level editor for the Titan Game Engine.

Originally forked from Sledge, renamed to Titan Editor as per the original license requirements.

The Titan Editor is an open-source level editor written in .NET and uses [Veldrid][2] for rendering, with DirectX 10.

Titan Editor supports map editing and is integrated with the Titan Game Engine.

The Titan Editor is licensed under BSD-3.
